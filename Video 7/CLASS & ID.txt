Difference Between Class and ID in HTML & CSS

1. Definition:

* Class: A class is used to group multiple elements so that they can share the same styles.
* ID: An id is used to uniquely identify a single element on a webpage.

2. Usage:

* Class can be used on multiple elements.
* ID should be used only once per page.

3. Syntax:

* Class is written with a dot (.)
  Example: .box { color: red; }

* ID is written with a hash (#)
  Example: #header { color: blue; }

4. HTML Example:

<div class="box">This is a class example</div>
<div class="box">Another class example</div>

<div id="header">This is an ID example</div>

5. Key Difference:

* You can assign the same class to multiple elements.
* You should not assign the same id to more than one element.

Conclusion:
Class is used for styling multiple elements, while ID is used for targeting a single unique element.
