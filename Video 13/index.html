CSS Are Three Types:-

  1. Inline CSS
  2. Internal CSS
  3. External CSS
Inline CSS:-
Inline CSS is the styling applied directly to an HTML element using the style attribute, affecting only that specific element.
Internal CSS:-
Inline CSS is the styling written directly within an HTML element using the `style` attribute.
External CSS:-
External CSS is the styling written in a separate .css file and linked to an HTML document using the tag, allowing styles to be applied across multiple pages.
