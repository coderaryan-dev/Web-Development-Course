## Semantic Tags (HTML)

Semantic tags are HTML elements that clearly describe the meaning and structure of content to browsers and developers.

### Common Semantic Tags

* `<header>` – Defines the top section (logo, title, navigation)
* `<nav>` – Contains navigation links
* `<main>` – Main content of the page
* `<section>` – Groups related content
* `<article>` – Independent content (blog, post)
* `<aside>` – Side content (sidebar)
* `<footer>` – Bottom section (copyright, links)

### Example

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Semantic Example</title>
</head>
<body>

  <header>
    <h1>My Website</h1>
  </header>

  <nav>
    <a href="#">Home</a>
    <a href="#">About</a>
  </nav>

  <main>
    <section>
      <article>
        <h2>Blog Title</h2>
        <p>This is a blog post.</p>
      </article>
    </section>

    <aside>
      <p>Sidebar content</p>
    </aside>
  </main>

  <footer>
    <p>© 2026 My Website</p>
  </footer>

</body>
</html>
```
