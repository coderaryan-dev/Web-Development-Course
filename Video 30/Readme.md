## 🧠 CSS Selectors Covered

### 🔹 Universal Selector

Applies styles to all elements:

```
*
```

### 🔹 Class Selector

Targets elements with a specific class:

```
.card
```

### 🔹 Element Selector

Targets specific HTML tags:

```
h2
```

### 🔹 Descendant Selector

Targets elements inside another element:

```
.card p
```

### 🔹 Pseudo Classes

Used for interaction:

```
:hover
:first-child
:last-child
:nth-child()
```
